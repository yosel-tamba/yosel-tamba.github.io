Virtual Multiple Skill Laboratory

Codeigniter Bootstrap

Virtual Multiple Skill Laboratory (V-MSL) is an educational application that provides online class and online quiz features. Through V-MSL, users can participate in online learning, access lesson materials, and take quizzes to test their understanding. This application is designed to support an interactive and flexible teaching and learning process. In addition, V-MSL offers a user-friendly interface that makes it easy for both students and teachers to navigate and utilize its features. The platform also enables teachers to create and manage classes, assign tasks, and monitor student progress in real time. With its comprehensive set of tools, V-MSL aims to enhance the overall educational experience and foster effective digital learning environments.

Pustaka 7

Codeigniter Bootstrap

Pustaka 7 is a digital school library application designed to facilitate both students and teachers in borrowing digital books. The platform provides an easy-to-use interface for searching, borrowing, and reading books online. In addition, Pustaka 7 features an admin dashboard that allows administrators to manage users and books efficiently. This comprehensive system aims to support digital literacy and streamline library management within the school environment.

Attendance System Website

Laravel React Bootstrap

This is a student attendance management website designed for use in various schools, and it does not have a specific name to allow for flexible implementation. The platform features attendance monitoring that can be performed by homeroom teachers, as well as student attendance input through student card tapping. Additionally, administrators have access to a dashboard for managing attendance records, homeroom teachers, and students. The application also includes a school information management feature, allowing each school to configure its own details and settings, making the attendance website highly flexible and adaptable for different educational institutions.

Evalingo

Codeigniter Bootstrap

Evalingo is a language learning application that combines AI assistance and gamification to create an engaging learning experience. Users can access learning materials, take quizzes, and compete on a ranking scoreboard, making the process interactive and fun like playing a game. The platform features an AI assistant powered by the Llama 3.2 3B large language model, allowing users to ask questions about any material they find challenging. This AI-driven support helps users better understand the content and enhances their overall learning journey.
